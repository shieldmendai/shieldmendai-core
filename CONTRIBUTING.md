# Contributing

Thank you for helping improve ShieldMendAI.

## Guidelines

- Keep repairs safe and reversible
- Create backups before changes
- Never hardcode secrets
- Document new features
- Test before submitting changes

## Pull Requests

1. Fork repository
2. Create a feature branch
3. Make changes
4. Test changes
5. Submit pull request

## Reporting Issues

Please include:

- Operating system
- Error messages
- Relevant logs
- Steps to reproduce
- Expected behavior
